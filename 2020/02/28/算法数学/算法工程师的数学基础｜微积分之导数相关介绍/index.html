按位计算的向量函数及其导数

假设一个函数$f(x)$的输入是标量$x$。对于一组$K$个标量$x_1, x_2,…,x_K$，我们可以通过$f(x)$得到另外一组$K$个标量$z_1, z_2,…,z_K$

为了简单起见，我们定义$x = [x_1, …, x_K]^T$, $z =[z_1, …, z_K]$

其中，$f(x)$是按位运算的，即$[f(x)]_i = f(x_i)$

当 $x$为标量时，$f(x)$的导数记为$f’(x)$。当输入为$K$维向量$x=[x_1, …,x_K]^T$时，其导数为一个对角矩阵。
